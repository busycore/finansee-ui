
# FinanSee UI (ReactJS)

FinanSee UI just a simple ReactJS Application handcrafted to help people finances' management.
## 📄 Requirements

```bash
NodeJS >= 14
FinanSee API previously running
```

## ⚙️ Configuration

```bash
.env

REACT_APP_API_URL=3005 # Backend API port 
```

## 🟢 Running the app

```bash
#install dependencies
$ npm install
# development
$ npm run start
```
